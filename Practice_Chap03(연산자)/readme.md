## 알아야 할 내용만 정리
### Operation Default Type
  1. 연산시에 int보다 작은 타입은 int로 변환 후 연산을 함 : char, byte, short 
  2. 정수형, 실수형 연산 시에는 표현 범위가 넓은 쪽으로 형변환 
      * (ex) byte + float => float + float = float / int + double => double + double = double
